# 여행 기록 앱

본 프로젝트는 여행지에서의 소중한 기록을 사진과 위치 정보 기반으로 저장하고 관리하는 안드로이드 애플리케이션입니다.

## 프로젝트 소개
- **주요 기능**:
    - **여행 기록 CRUD**: 사진, 장소, 날짜, 내용을 포함한 여행 기록 작성 및 관리.
    - **지도 기반 시각화**: 사진에 포함된 위치 정보를 추출하여 지도(Google Maps)에 마커로 표시.
    - **스마트 검색**: 장소나 키워드 기반으로 여행 기록 검색.

## 기술 스택
- **Language**: Kotlin
- **Libraries**: Google Maps API, Room/SQLite, Coroutines
- **Dev Tools**: Android Studio

## 테스트 및 검증 참고사항

**본 앱은 사진의 EXIF 메타데이터(위도/경도)를 기반으로 지도 마커를 생성합니다.**

- **위치 정보가 포함된 사진의 경우**: 카메라 설정에서 '위치 태그 저장'이 활성화된 사진을 등록하면 지도상에 마커가 정상적으로 생성됩니다.
- **위치 정보가 없는 사진의 경우**: EXIF 데이터가 없는 사진은 보안 및 데이터 무결성을 위해 무분별한 마커 생성을 방지하고 있습니다.
- **테스트 결과**: 에뮬레이터 환경 및 위치 정보가 포함된 원본 사진으로 테스트를 완료하였습니다. 실제 기기 테스트 시 위치 태그가 있는 사진을 사용하시면 정상적으로 마커가 노출되는 것을 확인하실 수 있습니다.
